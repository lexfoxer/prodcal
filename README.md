# prodcal_ics

Генератор `.ics` (iCalendar) с выходными и предпраздничными днями по данным производственного календаря.

Источник данных: страница календаря на hh.ru вида `https://hh.ru/article/calendar2026`.

## Требования

- Bun (проект использует Bun как рантайм и пакетный менеджер)

## Установка

```bash
bun install
```

## Запуск

Сгенерировать календарь за конкретный год:

```bash
bun run index.ts --year 2026
```

Также можно передать год позиционным аргументом:

```bash
bun run index.ts 2026
```

Если год не указан, будет использован текущий год.

## Результат

После запуска в корне проекта появится файл:

- `${year}.ics` (например, `2026.ics`)

Его можно импортировать в календарь (Google Calendar / Apple Calendar и т.п.) как обычный `.ics`.
